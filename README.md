# bc-eyeva-docs
Documentation for the Eyeva BigCommerce Theme by PapaThemes

## Installation

Run as administrator:

```
pip install mkdocs-material
pip install mkdocs-macros-plugin
```


## Development

Run command on terminal:

```
mkdocs serve
```

## Deployment

Run command on terminal:

```
mkdocs gh-deploy --clean --force
```
